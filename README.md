# SmartBuy Supermarket Billing System

## Project Description
The **SmartBuy Supermarket Billing System** is a Python-based application developed to automate supermarket billing operations. The system allows cashiers to enter product information, calculate totals, apply discounts, and generate receipts efficiently.

---

## Features
* **Product Entry System:** Easily input product names, quantities, and prices.
* **Smart Calculation:** Automated quantity and price computation.
* **Automatic Discounting:** Applies a **10% discount** automatically for total bills exceeding **Le 500**.
* **Receipt Generation:** Prints a clean, formatted receipt for the customer.
* **Continuous Workflow:** Supports multiple customer processing in a single session.
* **Data Storage:** Leverages Python lists for efficient in-memory data management.

---

## Technologies Used
* **Language:** Python 3.x

---

## How to Run the Program

1. **Install Python 3:** Ensure you have Python 3 installed on your machine.
2. **Download the Files:** Clone or download this repository to your local system.
3. **Open Terminal:** Open your terminal, command prompt, or git bash.
4. **Execute the Script:** Navigate to the project directory and run the following command:

```bash
python supermarket_billing_system.py

```
